## ğŸš€ Getting Started
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/artificial-intelligence-projects.git
   ```
2. **Navigate to the project folder**:
   ```bash
   cd artificial-intelligence-projects
   ```
3. **Install dependencies** (for practicals):
   - Ensure **Python 3.11** is installed.
   - Install required libraries:
     ```bash
     pip install numpy matplotlib scikit-learn
     ```
   - For Teachable Machine, use the online tool; no local install needed.
   - Run Python scripts:
     ```bash
     python script.py
     ```

4. **Explore the assignments**:
   - Each assignment folder contains documents (.pdf/.docx), Python code (.py) where applicable, README with objectives, instructions, and outputs.
   - Theoretical assignments include Word/PDF reports; practicals include code and screenshots.

## ğŸ§ª Assignments Overview
- **[Theory Assignment 1: AI The Past and the Present](assignments/theory_assignment1_ai_history/)**  
  Historical events in AI, advantages/disadvantages of GPS/LISP, current AI applications, and challenges.

- **[Theory Assignment 2: Search Problems](assignments/theory_assignment2_search_problems/)**  
  Formulate search problems for campus navigation, event planning, library book retrieval; apply BFS and IDDFS on a graph.

- **[Theory Assignment 3: A* Search](assignments/theory_assignment3_astar_search/)**  
  Traverse a graph using A* with straight-line distance heuristic.

- **[Practical Assignment 1: Teachable Machine](assignments/practical_assignment1_teachable_machine/)**  
  Train an ML model to recognize an object/action, analyze data collection and functionalities.

- **[Practical Assignment 2: Clustering Random Data](assignments/practical_assignment2_clustering_random_data/)**  
  Generate artificial data, apply KMeans clustering, visualize results.

- **[Practical Assignment 3: Clustering Fitness Tracker Data](assignments/practical_assignment3_clustering_fitness_tracker/)**  
  Simulate fitness data, cluster using KMeans, analyze user groups.

- **[Practical Assignment 4: FOL-Based Healthcare Chatbot](assignments/practical_assignment4_fol_chatbot/)**  
  Implement a diagnostic chatbot using first-order logic rules in Python.
